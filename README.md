# Detección y Clasificación Automática de Levaduras

Este proyecto presenta el diseño e implementación de un sistema de visión por computadora para la detección y clasificación automática de levaduras en imágenes microscópicas, orientado al análisis de viabilidad en procesos cerveceros.

## Descripción

Se desarrollaron modelos de clasificación de imágenes basados en redes neuronales convolucionales (CNNs), optimizados para lograr alta precisión y robustez frente a variaciones en las condiciones de captura. El sistema contempla un pipeline completo que incluye:

- Preprocesamiento de imágenes microscópicas  
- Entrenamiento de modelos de deep learning  
- Evaluación y validación de resultados  
- Clasificación automática de levaduras  

El prototipo fue desarrollado en colaboración con investigadores del CONICET y el equipo de microbrew.ar.

## Aplicaciones

El sistema permite automatizar tareas de análisis microbiológico, reduciendo tiempos y mejorando la consistencia en la evaluación de muestras, con potencial aplicación en la industria cervecera y laboratorios de investigación.
